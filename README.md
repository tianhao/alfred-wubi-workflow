# 安装：

* 安装图片处理工具imagemagick： `brew install imagemagick`
* 安装本 workflow

# 使用:

* wb 汉字
* 快捷键(⌃+⌘+C) -> 汉字
* 选中汉字->按快捷键(⌃+⌘+C)
